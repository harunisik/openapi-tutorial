The ISIN code ensures a uniform and globally accepted standard for identifying securities, which facilitates efficient trading, clearing, and settlement processes across different markets and systems.Structure of an ISIN1. Country Code (2 characters): The first two characters represent the country where the issuing entity is headquartered, according to ISO 3166-1 alpha-2 country codes. For example, "US" for the United States, "GB" for the United Kingdom.2. National Security Identifier (9 characters): The next nine characters are a unique identifier assigned to the specific security by the national numbering agency (NNA) of the issuing country. This part often includes letters and numbers and can be based on existing local identification codes.3. Check Digit (1 character): The final character is a check digit, which is used to verify the accuracy of the ISIN. It is calculated using the Luhn algorithm, ensuring the ISIN is correctly formed and valid.Bloomberg TickerBloomberg ticker symbols are a unique identifier system used by Bloomberg L.P. to provide an easy and consistent way to reference securities in the Bloomberg Terminal. These symbols are structured to provide immediate and clear information about the security type, market, and specific characteristics.down below is a description on the different components of the BB Ticker.Structure of an BB Ticker1. Base Symbol: The base symbol typically represents the issuing company or security name. This is usually an abbreviation or a recognizable shorthand. For example, "AAPL" for Apple Inc.2. Exchange Code: Following the base symbol, an exchange code indicates where the security is traded. This code consists of one to four characters. For instance, "US" for securities traded on a U.S. exchange, or "LN" for the London Stock Exchange.3. Security Identifier: Additional characters or codes may be added to indicate specific types of securities, such as common stock, preferred stock, or bonds. For instance, a common stock might have a ticker like "AAPL US Equity," while a bond might have a different identifier attached to the base symbol.Security Identifier List:most common for us:* EQUITY - common�shares* COMDTY - commodity markets�* INDEX - indices* CRNCY - currency marketsless common:* GOVT - government securities (US treasury and non-US)* CORP - corporate debt* MTGE - mortgage securities* M-Mkt - money market, commercial paper* MUNI - municipal debt* PFD - preferred shares* PORTF- portfolio and risk analysis
